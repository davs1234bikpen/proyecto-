Python

Python es ampliamente considerado como un lenguaje de programación fácil de aprender,
debido a su sintaxis sencilla, la gran biblioteca de estándares, herramientas e integración con
otros lenguajes de programación populares como C y C++.Es un lenguaje de programación
popular, especialmente entre los nuevos emprendimientos y, por lo tanto, las habilidades en
Python tienen una gran demanda. Inconvenientes: Python no es adecuado para el desarrollo
de aplicaciones móviles.Se utiliza en una amplia variedad de aplicaciones, incluida la
inteligencia artificial, los servicios financieros y la ciencia de datos. Páginas de redes sociales
como Instagram y Pinterest también se basan en Python.

Algunas razones por las que é escogido Python son :

 Sintaxis clara y legible: Python se destaca por su sintaxis sencilla y legible, lo que facilita la comprensión del código. Su enfoque en la legibilidad fomenta el desarrollo de un código limpio y estructurado, lo que facilita el mantenimiento y la colaboración en proyectos

Amplia comunidad y abundante documentación: Python cuenta con una gran comunidad de desarrolladores activos que brindan soporte y comparten recursos. Esto significa que puedes encontrar una gran cantidad de bibliotecas, módulos y documentación en línea para ayudarte a resolver problemas y acelerar el desarrollo de tu proyecto.

JavaScript

JavaScript es el lenguaje de programación más popular para crear sitios web interactivos; cuando
se combina con Node.js, los programadores pueden usar JavaScript para producir contenido web
en el servidor antes de enviar una página al navegador, de este modo creando juegos y aplicaciones
de comunicación que se ejecutan directamente en el navegador. Existe una amplia variedad de
complementos que también amplían la funcionalidad de JavaScript. Inconvenientes Los navegadores
de Internet pueden deshabilitar la ejecución del código JavaScript, ya que JavaScript se usa para
codificar anuncios emergentes que, en algunos casos, pueden contener contenido malicioso.

Algunas razones por las que é escogido JavaScript son :

Fácil aprendizaje: JavaScript es un lenguaje relativamente fácil de aprender, especialmente para aquellos que tienen conocimientos previos de programación. La sintaxis de JavaScript es similar a otros lenguajes de programación, lo que facilita la transición.

Versatilidad: JavaScript es un lenguaje versátil que se puede utilizar para desarrollar una amplia gama de aplicaciones, desde aplicaciones web interactivas y dinámicas hasta aplicaciones móviles híbridas utilizando frameworks como React Native o Ionic.

Amplia compatibilidad: JavaScript es un lenguaje de programación que se ejecuta en el navegador, lo que significa que es compatible con prácticamente todos los navegadores web modernos. Además, también puede ser utilizado en el lado del servidor a través de plataformas como Node.js.
